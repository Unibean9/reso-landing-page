# scrollxuidesign-aisaas_template [![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FAdityakishore0%2Fscrollxuidesign-aisaas_template)

Built with next.js, [shadcn/ui](https://ui.shadcn.com/), and [scrollx ui](https://scrollxui.dev/).

# Features

- Setup only takes a few minutes by editing the files 
- Built using Next.js 16, React, Typescript, Shadcn/UI, TailwindCSS, Motion, ScrollX UI
- Responsive for different devices
- Optimized for Next.js and Vercel

# Getting Started Locally

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/Adityakishore0/scrollxuidesign-aisaas_template
   ```

2. Move to the cloned directory

   ```bash
   cd scrollxuidesign-aisaas_template
   ```

3. Install dependencies:

   ```bash
   pnpm install
   ```

4. Start the local Server:

   ```bash
   pnpm dev
   ```

5. Open the files and sections and make changes
